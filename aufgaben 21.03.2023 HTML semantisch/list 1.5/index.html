ul vs. ol

  • unordered list
    • HTML-tag:ul
    • listet Punkte mit bullet point auf
    • Beispiele:
      • To-DO Liste
      • ...
      • ...

    • Weiterführende Infos

  • ordered list
    • HTML-tag: ol
    • listet punkte mit Nummerierungen auf
    • Beispiele:
      1. Inhaltsverzeichnis
      2. ...
      3. ...

    • Weiterführende Infos
